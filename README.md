**Installation**
---
`$ npm install`

**Running app in dev**
---
`$ npm run dev`

**Running tests**
---
`$ npm test`

**Check lint**
---
`$ npm run lint`

**Check Prettier**
---
`$ npm run prettier`

**Creating migration**
---
`$ npm run migration:generate nameMigration`

**Running migrations**
---
`$ npm run migration:run`
